# Development Roadmap (.agent/ROADMAP.md)

이 문서는 주식 정보 플랫폼 구축을 단계별 마일스톤과 구체적인 하위 체크리스트로 구분하여 기록한 자율적 개발 로드맵입니다. 향후 기능이 점진적으로 추가될 수 있도록 하단에 빈 여유공간(Extension Backlogs)을 포함하고 있습니다.

---

## [Milestone 1] 개발 기초 다지기 & 환경 구축
*백엔드와 프론트엔드의 견고한 기술 기초 환경과 기초 연동을 검증하는 첫 번째 마일스톤입니다.*

- [x] **1.1. 프론트엔드 인프라 빌드 환경 세팅**
  - [x] Vite React 프로젝트 확인 및 `package.json` 클렌징
  - [x] TailwindCSS v3, PostCSS, Autoprefixer 패키지 완벽 통합
  - [x] `tailwind.config.js` 및 `postcss.config.js` 파일 생성 및 토스 포인트 디자인 컬러 바인딩
  - [x] `main.jsx`에 `index.css`를 포함하고 컴파일 테스트 완료
- [x] **1.2. 백엔드 Spring Boot 3 & PostgreSQL 설정**
  - [x] Java 21 기반의 Gradle 프로젝트 동작 유효성 검증
  - [x] `application.properties` 또는 `application.yml`에 PostgreSQL Supabase 커넥션 설정 준비
  - [x] 루트 경로에 민감정보 관리를 위한 `.env` 뼈대 파일 및 `.gitignore` 설정 연동
  - [x] JPA의 `ddl-auto=update` 옵션 작동 검증 및 롬복 라이브러리 테스트
- [x] **1.3. 통합 테스트 스켈레톤 구축**
  - [x] 백엔드 빌드 유효성 테스트 (`./gradlew clean build`)
  - [x] CORS 교차 출처 리소스 공유 허용 범위 정의 (`*` 또는 프론트엔드 포트 포인팅)

## [Milestone 2] JWT 기반 회원 관리 및 보안 인프라
*플랫폼 전체에 걸쳐 안전하게 사용자 식별을 할 수 있는 JWT 인증 체계를 백엔드와 프론트엔드 양측에 구현합니다.*

- [x] **2.1. JWT 토큰 발급 & 파싱 모듈 구현**
  - [x] JJWT 라이브러리를 사용해 `JwtTokenProvider` 개발 (보안 대칭키 기반 암호화)
  - [x] JWT 생성 시간, 만료 시간(24시간) 설정 로직 구성
- [x] **2.2. Spring Security 필터 체인 체계 확립**
  - [x] Stateless(무상태) 세션 정책 구성 및 CSRF 방지 비활성화
  - [x] `JwtAuthenticationFilter`를 만들어 `UsernamePasswordAuthenticationFilter` 전단에 삽입
  - [x] 회원가입/로그인/주식조회 등 비인증 접근 허용 목록(`requestMatchers`) 명시
- [x] **2.3. 회원 인증 API 및 로직 개발**
  - [x] `tb_users` 테이블에 연동되는 `MemberDTO` (Entity) 및 Repository 구현
  - [x] BCrypt 해시 인코더를 활용한 패스워드 단방향 암호화 가입 기능 (`/api/member/join`)
  - [x] 로그인 성공 시 JWT 토큰 및 핵심 메타데이터 반환 API 구현 (`/api/member/login`)
  - [x] HTTP Bearer 헤더 기반 내 프로필 조회 API 구현 (`/api/member/profile`)
  - [x] **[신규]** 회원 테이블 비대화 방지를 위한 1:1 `MemberProfileDTO` (프로필 엔티티) 양방향 매핑 설계 및 테이블 구축 완료
  - [x] **[신규]** 회원가입 시 물리적 프로필 이미지 첨부 파일 업로드 및 D드라이브 부재 시 워크스페이스 상대경로(`upload/members/`) 자동 백업 폴백 안전장치 구현 완료
- [x] **2.4. 프론트엔드 인증 연동 컴포넌트 개발**
  - [x] `/utils/api.js` 공통 fetch 모듈 구현 (로컬스토리지 JWT 자동 탑재 기능 포함)
  - [x] `/auth` 페이지 구현: 토스 특유의 둥근 모서리 로그인/회원가입 탭 토글 모던 카드 디자인
  - [x] 로그인 후 Context/State를 활용한 내비게이션바의 로그인 상태 표시 동적 전환
  - [x] **[신규]** 회원가입 폼 전송 방식을 일반 JSON에서 `FormData` 방식으로 전환하고 프로필 이미지 업로드 파일 인풋 컴포넌트 연동 튜닝 완료

## [Milestone 3] 토스 스타일 주식 대시보드 & 인터랙티브 차트
*인터페이스 기반으로 유연하게 결합된 주식 공급자 모듈과, 수집된 데이터를 미려하게 시각화하는 프론트엔드 라인 차트를 개발합니다.*

- [x] **3.1. 느슨한 결합의 주식 도메인 API 설계**
  - [x] `Stock` 및 `StockHistory` 테이블과 1대N 연동되는 JPA 데이터 엔티티 매핑
  - [x] `StockService` 인터페이스 정의 (향후 한국투자증권 API / MCP 서버로의 원활한 변경 목적)
  - [x] DB 레코드와 난수 증감을 활용해 실제 장 거래처럼 동작하는 고성능 `MockStockServiceImpl` 구현
- [x] **3.2. 실시간 가격 흐름 및 히스토리 API 개발**
  - [x] 현재 등록된 인기주식/종목 목록 조회 API (`/api/stock/list`)
  - [x] 특정 주식 종목의 기간별(7일, 30일) 상세 히스토리 조회 API (`/api/stock/detail/{symbol}`)
- [x] **3.3. 인터랙티브 SVG 주식 라인 차트 구현**
  - [x] 라이브러리를 배제하고 TailwindCSS 디자인 토큰과 SVG Primitives만으로 반응형 차트 컴포넌트 (`StockChart.jsx`) 구현
  - [x] 마우스/터치 오버 시 세로 보조선 표시 및 해당 날짜 가격 수치 툴팁 출력 연동
  - [x] 전일 대비 가격의 등락에 따른 색상 동적 전환 (한국 주식 표준 빨강/파랑 테마 연동)
- [x] **3.4. 메인 주식 대시보드 페이지 (`/`) 구축**
  - [x] 토스 홈과 유사한 카드형 레이아웃 배치
  - [x] 주요 마켓 인덱스(Kospi, Kosdaq) 및 상승률 상위 종목, 인기 검색 종목 실시간 연동 리스트 탑재

- [x] **4.1. 자유게시판(FreeBoard) CRUD API 개발**

  - [x] 조회수 증가 방지 가드 및 추천(좋아요) 증감 트랜잭션 구현
  - [x] 제목/본문 키워드 통합 검색 쿼리 API 추가
- [x] **4.2. 실시간 게시판 댓글(Comment) 시스템 구축**

  - [x] 게시글 상세 페이지 진입 시 해당 댓글 목록 로드 및 작성/삭제 API 연동
